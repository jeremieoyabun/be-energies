# Information Architecture Skill

Create the complete information architecture for the new Be'energies site.

## Must include
- homepage
- about page
- services hub
- individual service pages
- realizations index
- realization detail pages
- "Pièges à éviter" pillar page
- blog index
- article templates
- contact page
- pro page
- local city-service landing pages
- bilingual FR/NL route strategy
- legal pages
- optional lead magnet landing pages

## Constraints
- SEO-first
- conversion-first
- avoid duplicate intent
- scalable to 30 to 50 local pages
- internal linking must be intentional
- mobile-first nav logic

## Output
1. final sitemap
2. URL conventions
3. page purpose
4. primary keyword intent
5. conversion goal per page
6. reusable sections
7. internal linking map
8. breadcrumb logic
