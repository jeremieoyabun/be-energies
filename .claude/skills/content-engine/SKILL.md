# Content Engine Skill

Use the conversion-copywriter subagent.

## Write content for
- homepage
- about page
- each service page
- local landing pages
- realizations intros
- "Pièges à éviter" pillar page
- FAQs
- lead magnet pages
- blog articles

## Content rules
- French for Belgium first
- concrete, credible, not fluffy
- founder expertise must appear naturally
- explain real mistakes, risks, and quality choices
- use trust and proof often
- write for humans first, SEO second
- maintain distinct search intent by page type

## For every page deliver
- SEO title
- meta description
- H1
- hero copy
- full section structure
- CTA copy
- FAQ copy
- internal link suggestions
- schema suggestions when relevant
