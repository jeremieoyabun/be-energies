# Project Bootstrap Skill

Your task is to initialize this website rebuild project from the provided crawl, business context, and goals.

## Objectives
- transform raw project context into an execution-ready roadmap
- identify missing assumptions
- define phases, deliverables, and dependencies
- create the initial project plan before implementation

## Steps
1. Read CLAUDE.md fully.
2. Summarize the business, offer, audience, geographies, and differentiators.
3. Extract the current site structure and its weaknesses.
4. Define a recommended future-state architecture:
   - page types
   - data types
   - reusable sections
   - content systems
   - multilingual needs
5. Create a phased implementation roadmap.
6. Output a prioritized backlog with:
   - P0
   - P1
   - P2
7. Identify what should be static, generated, CMS-driven, or manually curated.
8. Propose the initial folder structure for the Next.js app.

## Output format
- Executive summary
- Key strategic decisions
- Architecture proposal
- Phase plan
- Backlog
- Risks and mitigation
