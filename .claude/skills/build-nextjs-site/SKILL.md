# Build Next.js Site Skill

Use the nextjs-architect subagent.

Build a production-grade website for Be'energies.

## Stack
- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- content collections or CMS-ready architecture
- Vercel-ready
- structured data utilities
- optimized images
- route generation for local pages
- bilingual-ready architecture

## Build requirements
- homepage
- service pages
- about page
- local landing system
- realizations pages
- blog pages
- contact
- legal pages
- shared components
- metadata
- schema

- robots

- preload only critical assets
- no unnecessary dependencies
- optimize font loading
- ship clean semantic markup

- all pages and components
- sample content data
- utility functions
- metadata strategy
- schema helpers
- launch-ready codebase
